越狱第三章：企业微信抢红包插件

企业微信抢红包（为年终奖做准备( ⁼̴̀ .̫ ⁼̴́ )✧）

使用logly.pl自动编写打印方法的文件

  1. /opt/theos/bin/logify.pl /文件目录/WWKConversationRedEnvelopesBubbleView.h > ./1.xm
  2. 如果要跟踪多个头文件，则继续转换：/opt/theos/bin/logify.pl 其它头文件 >> Tweak.xm    注意中间是 >>（没试过）

安装到手机

export THEOS=/opt/theos
export THEOSDEVICEIP=192.168.1.100
先执行以上两行命令
然后执行
make package install
make package 和 make install可以单独使用

make package报错

有的时候会报错，不知道类名，这种情况在文件最上面写上

@interface WWKMessageRedEnvelopes
@end

写上类似此种格式的代码，骗过编译
